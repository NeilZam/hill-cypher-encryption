# Hill Cipher Encryption

A cryptography application written in C that encrypts text using the Hill Cipher algorithm and matrix operations.

## Features

- Matrix-based encryption
- Variable key sizes
- File input/output
- Plaintext processing
- Ciphertext generation

## Concepts Used

- Cryptography
- Matrix Operations
- Modular Arithmetic
- File Processing

## Technologies

- C
- GCC
- Command Line

## What I Learned

This project introduced classical encryption techniques and strengthened my understanding of matrix-based algorithms.
